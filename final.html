Artist's Songs on the Billboard Charts

[ made by Chloe Peng - Period 5 ]


Project Overview

This is a python program that will show the songs of an artist (typed in by the user) that have made it onto the Billboard's Hot 100's songs from the years 2000 - 2021


Instructions

After running the program, it'll ask you to input an artist (singer, band, group etc.) name. Make sure to spell it exactly as the way they're known by (like capitalization and spacing)!


How does it Work?

By using the input() python program
