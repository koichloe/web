Artist's Songs on the Billboard Charts

[ made by Chloe Peng - Period 5 ]


Project Overview

This is a python program that will show the songs of an artist (typed in by the user) that have made it onto the Billboard's Hot 100's songs from the years 2000 - 2021. By using a data set of all the songs that have been on the weekly Billboard Hot 100's Charts since 1958, the python code and analysis and read through the data set and determine the songs of an artist that were on there. The data set includes a lot of information but it doesn't include the highest position a song has ever reached on the charts. Which is why I included a way it could determine each song's peak position on the charts in my python code.


Instructions

After running the program, it'll ask you to input an artist (singer, band, group etc.) name and then click enter. Make sure to spell it exactly as the way they're known by (and same with capitalization / spacing)!


How does it Work?

By using the input() function in python, the user is able to input an artist of their choice. And after the program acquires the name of an artist, it goes through multiple functions to find the songs of that artist in the data set. I created a modified list function that cuts the orignal data set into a list of lists that includes all the information of each songs that were on the charts from the 2000's as an element of that list. And then another function searches through the modifed list to find every list in that list that has a song by the artist. Then, it reorganizes the lists of every time a song by the artist has been on the charts into a dictionary where the keys are all the songs that's appeared and the values is a list of all the position the song has ever been on the charts. Then, it changes the values of the dictionary into a list of only the highest position the song has ever been on the charts by running an outside function that would determine the smallest value in a list. And the final part of the program prints out the name of an aritst and then it's peak position.
